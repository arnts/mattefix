2.1 Førstegradsligninger

Regning med bokstaver

Når vi regner med bokstaver, skriver vi leddene i alfabetisk rekkefølge:

Riktig rekkefølge
ab,   2xy,   3abc
Ikke skriv
ba,   2yx,   3cba

Tall skrives foran bokstavene, og vi dropper gangetegnet:

a + a + a = 3a     a · b = ab     2 · x · y = 2xy

Algebraiske uttrykk

Et algebraisk uttrykk inneholder tall, bokstaver og regneoperasjoner. Vi forenkler ved å samle like ledd:

3x + 2x − 5 + 7 = 5x + 2
Samle like ledd
3x + 2x = 5x
Like ledd har samme bokstavdel
Distributiv lov
a(b + c) = ab + ac
3(2x − 1) = 6x − 3
Parenteser med minus foran
−(a + b) = −a − b
−(2x − 5) = −2x + 5
Multiplisere parenteser
(a+b)(c+d) = ac+ad+bc+bd
(x+2)(x+3) = x²+5x+6

En ligning har et likhetstegn og sier at to uttrykk er like. Vi løser ligningen for å finne verdien av den ukjente.

Blokkmetoden (vektprinsippet)

En ligning er som en vekt i balanse. Det vi gjør på den ene siden, må vi også gjøre på den andre:

Vi kan addere, subtrahere, multiplisere eller dividere med det samme tallet på begge sider.

Eksempel 1: Løs 3x + 5 = 20

Trekk fra 5 på begge sider: 3x + 5 − 5 = 20 − 5   ⇒   3x = 15
Del på 3 på begge sider: 3x / 3 = 15 / 3   ⇒   x = 5
Kontroll: Sett inn x = 5: 3 · 5 + 5 = 15 + 5 = 20 ✓

Eksempel 2: Løs 2(x − 3) = 4x + 2

Ekspander venstre side: 2x − 6 = 4x + 2
Trekk fra 2x på begge sider: −6 = 2x + 2
Trekk fra 2 på begge sider: −8 = 2x
Del på 2: x = −4
Kontroll: V.S. = 2(−4 − 3) = 2(−7) = −14. H.S. = 4(−4) + 2 = −14 ✓

Eksempel 3: Løs 5x − 3 = 2x + 9

Samle x-ledd: Trekk fra 2x på begge sider: 3x − 3 = 9
Samle talledd: Legg til 3 på begge sider: 3x = 12
Del på 3: x = 4
Kontroll: V.S. = 5(4) − 3 = 17. H.S. = 2(4) + 9 = 17 ✓

2.2 Kvadratsetningene

Første kvadratsetning

(a + b)2 = a2 + 2ab + b2

Eksempel: (x + 3)2 = x2 + 2 · x · 3 + 32 = x2 + 6x + 9

Eksempel: (2x + 5)2 = (2x)2 + 2 · 2x · 5 + 52 = 4x2 + 20x + 25

Andre kvadratsetning

(a − b)2 = a2 − 2ab + b2

Eksempel: (x − 4)2 = x2 − 8x + 16

Eksempel: (3x − 2)2 = 9x2 − 12x + 4

Konjugatregelen

(a + b)(a − b) = a2 − b2

Eksempel: (x + 7)(x − 7) = x2 − 49

Eksempel: (3x + 4)(3x − 4) = 9x2 − 16

Identiteter

Kvadratsetningene og konjugatregelen er identiteter — de gjelder for alle verdier av a og b. En identitet er ikke en ligning vi løser, men en omskriving som alltid stemmer.

1. kvadratsetning
(a+b)2 = a2+2ab+b2
2. kvadratsetning
(a−b)2 = a2−2ab+b2
Konjugatregelen
(a+b)(a−b) = a2−b2
Vanlig feil: (a + b)2 ≠ a2 + b2. Man glemmer ofte dobbeltproduktet 2ab!

2.3 Fullstendig kvadrat

Å fullføre kvadratet betyr å skrive et andregradsuttrykk på formen (x + d)2 + e. Nøkkelformelen er:

x2 + bx = (x + b2)2 − (b2)2

Eksempel 1: Skriv x2 + 6x + 2 som fullstendig kvadrat.

Identifiser b: b = 6, så b/2 = 3.
Skriv om x2 + 6x: x² + 6x = (x + 3)² − 9
Legg til konstantleddet: x² + 6x + 2 = (x + 3)² − 9 + 2 = (x + 3)² − 7

Eksempel 2: Skriv x2 − 4x + 7 som fullstendig kvadrat.

Identifiser b: b = −4, så b/2 = −2.
Skriv om: x² − 4x = (x − 2)² − 4
Legg til konstantleddet: x² − 4x + 7 = (x − 2)² − 4 + 7 = (x − 2)² + 3
Hvorfor er dette nyttig? Fullstendig kvadrat brukes til å løse andregradsligninger, finne toppunkt/bunnpunkt til parabler, og er grunnlaget for å utlede løsningsformelen (abc-formelen).

2.4 Andregradsligninger

En andregradsligning har formen ax2 + bx + c = 0, der a ≠ 0.

Kvadratrot

Den enkleste typen: bare x2-ledd.

x2 = a   ⇒   x = ±√a   (når a ≥ 0)

Eksempel: x2 = 25  ⇒  x = ±5   (to løsninger: x = 5 og x = −5)

Eksempel: x2 = −4  ⇒  ingen reell løsning (kan ikke ta kvadratrot av negativt tall)

Løsning ved faktorisering

Hvis vi kan skrive uttrykket som et produkt av to faktorer, bruker vi produktregelen:

A · B = 0   ⇒   A = 0   eller   B = 0

Produktregelen (nullpunktregelen) sier at minst én av faktorene må være null for at produktet skal bli null.

Eksempel: Løs x2 − 5x + 6 = 0

Faktoriser: x² − 5x + 6 = (x − 2)(x − 3)
Sett lik null: (x − 2)(x − 3) = 0
Produktregelen: x − 2 = 0 ⇒ x = 2    eller    x − 3 = 0 ⇒ x = 3

Eksempel: Løs x2 − 9 = 0

Konjugatregelen: x² − 9 = (x + 3)(x − 3) = 0
Løsning: x = −3   eller   x = 3

Løsning ved sum og produkt

For ligningen x2 + bx + c = 0 leter vi etter to tall x1 og x2 slik at:

Summen av røttene
x1 + x2 = −b
Produktet av røttene
x1 · x2 = c

Eksempel: Løs x2 − 7x + 12 = 0

Finn to tall med sum 7 og produkt 12: 3 + 4 = 7   og   3 · 4 = 12 ✓
Løsning: x = 3   eller   x = 4
Faktorisering: x² − 7x + 12 = (x − 3)(x − 4)

2.5 Løsningsformel for andregradsligninger

For ligningen ax2 + bx + c = 0 gir abc-formelen:

x = −b ± √(b2 − 4ac)2a

Diskriminanten

Uttrykket under rottegnet kalles diskriminanten:

D = b2 − 4ac
D > 0
To løsninger
To ulike skjæringspunkter med x-aksen
D = 0
Én løsning
Dobbeltrot – tangerer x-aksen
D < 0
Ingen reelle løsninger
Grafen krysser ikke x-aksen

Eksempel: Løs 2x2 − 4x − 6 = 0

Identifiser a, b, c: a = 2,   b = −4,   c = −6
Regn ut diskriminanten: D = (−4)² − 4 · 2 · (−6) = 16 + 48 = 64
D > 0, altså to løsninger. Sett inn i formelen: x = (4 ± √64) / (2 · 2) = (4 ± 8) / 4
Finn de to løsningene: x = (4 + 8) / 4 = 3    eller    x = (4 − 8) / 4 = −1
Kontroll: 2(3)² − 4(3) − 6 = 18 − 12 − 6 = 0 ✓

Steg-for-steg: Hvilken metode?

Omform ligningen til formen ax2 + bx + c = 0.
Er b = 0? → Bruk kvadratrot direkte: x = ±√(−c/a).
Er c = 0? → Faktoriser ut x: x(ax + b) = 0.
Ser du faktorene? → Faktoriser og bruk produktregelen.
Ellers → Bruk abc-formelen. Sjekk diskriminanten D = b2 − 4ac for antall løsninger.

2.6 Praktisk bruk av andregradsligninger

Eksempel 1: Kast

Oppgave: En ball kastes opp. Høyden h (i meter) etter t sekunder er gitt ved h = −5t2 + 20t. Når er ballen 15 m over bakken?
Sett opp ligningen: −5t² + 20t = 15   ⇒   −5t² + 20t − 15 = 0
Del på −5: t² − 4t + 3 = 0
Faktoriser: (t − 1)(t − 3) = 0
Løsning: t = 1 s   eller   t = 3 s
Tolkning: Ballen er 15 m over bakken to ganger — på vei opp (t = 1 s) og på vei ned (t = 3 s).

Eksempel 2: Areal

Oppgave: Et rektangel har omkrets 20 cm og areal 24 cm2. Finn sidene.
Sett opp: La lengden være x. Omkrets = 2(x + b) = 20, altså b = 10 − x. Areal: x(10 − x) = 24
Omform: 10x − x² = 24   ⇒   x² − 10x + 24 = 0
Faktoriser: To tall med sum 10 og produkt 24: 4 og 6. (x − 4)(x − 6) = 0   ⇒   x = 4 eller x = 6
Svar: Sidene er 4 cm og 6 cm. (Begge løsningene gir samme rektangel.)
Tips: I praktiske oppgaver — sjekk alltid om begge løsningene gir mening. Negativ tid, negativ lengde eller umulige verdier forkastes.

2.7 Faktorisering og forkorting

Faktorisering av andregradsuttrykk

Når vi kjenner røttene x1 og x2, kan vi skrive:

ax2 + bx + c = a(x − x1)(x − x2)

Eksempel 1: Faktoriser x2 − 5x + 6

Finn røttene: Sum 5, produkt 6 → tallene 2 og 3. x² − 5x + 6 = (x − 2)(x − 3)

Eksempel 2: Faktoriser x2 − 4

Konjugatregelen: x² − 4 = x² − 2² = (x + 2)(x − 2)

Eksempel 3: Faktoriser 2x2 + 6x

Felles faktor: 2x² + 6x = 2x(x + 3)

Forkorting av brøkuttrykk

Faktoriser teller og nevner, og stryk felles faktorer:

Eksempel: Forkort x2 − 9x + 3

Faktoriser telleren med konjugatregelen: x² − 9 = (x + 3)(x − 3)
Forkort: (x + 3)(x − 3) / (x + 3) = x − 3   (for x ≠ −3)

Eksempel: Forkort x2 − x − 6x2 − 4

Faktoriser teller: x² − x − 6 = (x − 3)(x + 2)
Faktoriser nevner: x² − 4 = (x + 2)(x − 2)
Forkort (x + 2): (x − 3)(x + 2) / ((x + 2)(x − 2)) = (x − 3) / (x − 2)   (for x ≠ −2)

Ligninger med variabelen i nevner

Når x står i nevneren, må vi oppgi krav til x (nevneren kan ikke være null) og sjekke for falske løsninger.

Eksempel: Løs 1x − 2 + 3 = xx − 2

Krav: x ≠ 2 (nevneren kan ikke være null).
Gang begge sider med (x − 2): 1 + 3(x − 2) = x
Ekspander og løs: 1 + 3x − 6 = x   ⇒   3x − 5 = x   ⇒   2x = 5   ⇒   x = 2,5
Sjekk kravet: x = 2,5 ≠ 2 ✓ Løsningen er gyldig.
Viktig: Hvis løsningen du finner bryter med kravet (f.eks. x = 2 her), er det en falsk løsning og ligningen har ingen løsning.
