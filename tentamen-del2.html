Regresjon i GeoGebra

Regresjon brukes når du har en tabell med målinger og skal finne en modell (funksjon) som passer best. Du må også velge hvilken modell.

Steg for steg

Åpne regneark i GeoGebra (Vis → Regneark). Legg x-verdier i kolonne A, y-verdier i kolonne B
Marker tallene, høyreklikk → Lag → Liste med punkt. Da får du en punktliste (f.eks. L1) som vises i grafikkfeltet.
Se på formen på punktene:
  • Rett linje → lineær: RegLin(L1)
  • Kurve som vokser raskere og raskere → eksponentiell: RegEksp(L1)
  • Topp/bunn-kurve → polynom: RegPoly(L1, 2) (2 = andregrad)
  • Bare positive x, og y = 0 når x = 0 (f.eks. areal/volum vs. lengde) → potens: RegPot(L1)
Skriv opp funksjonen fra algebrafeltet, f.eks. f(x) = 2.4·1.08^x. Avrund fornuftig (3 gjeldende siffer er vanlig).
ALDRI bruk årstall som x-verdier! Tall som 2024, 2025 gir kjempestore eksponenter og ødelegger modellen. Bruk i stedet antall år etter et startår. Eksempel: år 2018 → x = 0, år 2024 → x = 6.

Eksempel: Innbyggere i en by

Du får tabellen:

År2018202020222024
Innbyggere5000540858496326

Sett år 2018 som x = 0. Da blir x-verdiene 0, 2, 4, 6.

# I regnearket: A1–A4 = 0,2,4,6 B1–B4 = 5000,5408,5849,6326 L1 = {(0,5000),(2,5408),(4,5849),(6,6326)} f(x) = RegEksp(L1) → f(x) ≈ 5000 · 1.04^x

Tolkning: vekstfaktoren 1.04 betyr at byen vokser med ca. 4 % per år. Etter 10 år: f(10) ≈ 7401 innbyggere.

Funksjonsanalyse i praktisk kontekst

Gjennomsnittlig vekstfart

Gjennomsnittlig vekstfart er stigningstallet til sekanten mellom to punkter på grafen.

Gj.snitt vekstfart = (f(b) − f(a)) / (b − a)

Tolkning: hvor mye funksjonen endrer seg i gjennomsnitt per enhet x mellom a og b.

Eksempel

f(x) = innbyggertall etter x år. f(0) = 5000 og f(6) = 6326.

Gj.snittlig vekst = (6326 − 5000) / (6 − 0) = 1326 / 6 ≈ 221 per år.

Tolkning: i gjennomsnitt kom det ca. 221 nye innbyggere per år i perioden.

Momentan vekstfart

Momentan vekstfart er stigningstallet til tangenten i ett bestemt punkt. Det er f'(x) = den deriverte.

Momentan vekstfart i x = a ⇔ f'(a)

Tolkning: endringsraten akkurat nå, altså hvor raskt f(x) endrer seg i punktet x = a.

# Finn momentan vekstfart i x = 5 f(x) = 5000 · 1.04^x f'(5) → ≈ 238.6

Definisjons- og verdimengde fra kontekst

  • Definisjonsmengde (Df): hvilke x-verdier gir mening? Negativ tid? Negativ lengde? Nei!
  • Verdimengde (Vf): hvilke y-verdier kan forekomme? Se på grafen.

Eksempel

En ball kastes. Høyde: h(t) = −5t² + 20t, der t er sekunder.

Ballen lander når h(t) = 0: t = 0 eller t = 4.

Dh = [0, 4] – fra kast til den lander. Maks høyde i t = 2: h(2) = 20. Vh = [0, 20].

Gyldighetsområde for modellen

En modell fungerer bare i et visst område. Utenfor dette blir tallene upålitelige.

  • Eksponentiell vekst for all framtid? Nei – byen kan ikke vokse mot uendelig.
  • Tredjegradsmodell for bremselengde? Bare mellom minste og største målte fart.
Huskeregel: Oppgi gyldighetsområdet innenfor måleperioden, eller si «modellen er gyldig fram til ca. år …». Ikke ekstrapoler langt utenfor.

Skjæringspunkter grafisk

Finn skjæring mellom to grafer med Skjæring(f, g) i GeoGebra. Bruk punktet til å svare på spørsmål som «når er tilbudet lik etterspørselen?» eller «når er de like høye?».

Polynomfunksjoner med ukjente parametere

Når du får en funksjon som f(x) = ax² + bx + c med tre ukjente, trenger du tre betingelser for å finne a, b og c. Hver betingelse blir én likning.

Typiske betingelser

Tekst i oppgavenLikning
Grafen går gjennom (2, 5)f(2) = 5
Grafen har nullpunkt i x = 3f(3) = 0
Topp-/bunnpunkt i x = 1f'(1) = 0
Gj.snittlig vekstfart fra 0 til 4 er 6(f(4) − f(0)) / 4 = 6
Tangent i x = 2 har stigningstall 3f'(2) = 3
Skjærer y-aksen i 7f(0) = 7

Løs med CAS

Definer funksjonen med ukjente:f(x) := a*x^2 + b*x + c
Skriv likningene som en liste.
Bruk Løs() med ukjente til slutt:Løs({likn1, likn2, likn3}, {a, b, c})
Sett inn verdiene i f(x) og skriv det endelige uttrykket.

Eksempel

Finn f(x) = ax² + bx + c når grafen går gjennom (0, 2), har bunnpunkt i x = 3, og f(5) = −3.

f(x) := a*x^2 + b*x + c f'(x) Løs({f(0) = 2, f'(3) = 0, f(5) = −3}, {a, b, c}) → {a = 1, b = −6, c = 2} # Endelig svar: f(x) = x² − 6x + 2

Kontroll: f(0) = 2 ✓, f'(3) = 2·3 − 6 = 0 ✓, f(5) = 25 − 30 + 2 = −3 ✓.

Pass på: I CAS skal du bruke := når du definerer f, og = i likningene inni Løs(). Variabler som a, b, c skal IKKE være definert som tall før du løser!

Rasjonale funksjoner

En rasjonal funksjon har formen f(x) = (teller) / (nevner), der både teller og nevner er polynomer.

f(x) = (ax + b) / (cx + d)

Asymptoter

  • Vertikal asymptote: der nevneren = 0 (og telleren ≠ 0). Grafen stikker av mot ± uendelig.
  • Horisontal asymptote: verdien grafen nærmer seg når x → ± uendelig. Les av fra grafen eller bruk regelen:
Grad teller vs. nevnerHorisontal asymptote
Teller < nevnery = 0
Teller = nevnery = (ledende koeffisient teller) / (ledende koeffisient nevner)
Teller > nevnerIngen horisontal asymptote (skrå i stedet)

Finne ukjente parametere

Har du f(x) = (ax + b) / (cx + d) og kjenner grafen:

  • Les av vertikal asymptote (for eksempel x = 2) → cx + d = 0 når x = 2, altså 2c + d = 0.
  • Les av horisontal asymptote (for eksempel y = 3) → a/c = 3.
  • Les av et punkt på grafen, f.eks. skjæring med y-aksen.
  • Sett opp likningene og løs med Løs() i CAS.

Eksempel

f(x) = (ax + b) / (x − 2) har horisontal asymptote y = 3 og skjærer y-aksen i 1.

  • Horisontal asymptote: a/1 = 3 → a = 3.
  • Skjærer y-aksen: f(0) = 1 → (a·0 + b) / (0 − 2) = 1 → b / (−2) = 1 → b = −2.

Svar: f(x) = (3x − 2) / (x − 2).

Kontroll: f(0) = (−2)/(−2) = 1. ✓

Sjekk alltid med y-aksen: Regn ut f(0) og sammenlign med det grafen viser. Dette er raskeste måte å oppdage slurvefeil.

Likningssett med to ukjente

Fra en tekstoppgave: finn to sammenhenger, sett opp to likninger, løs i CAS.

Fra tekst til likninger

Les oppgaven to ganger. Marker tall og størrelser som «antall», «pris», «kr», «kg».
Velg variabler (f.eks. x = antall epler, y = antall appelsiner) og skriv det ned.
Finn to sammenhenger. Ofte én for «antall» og én for «sum/pris».
Løs med CAS: Løs({likn1, likn2}, {x, y})

Eksempel

Per kjøper 3 epler og 2 pærer for 24 kr. Anna kjøper 2 epler og 5 pærer for 27 kr. Hva koster ett eple og én pære?

La x = pris på eple, y = pris på pære.

Løs({3x + 2y = 24, 2x + 5y = 27}, {x, y}) → {x = 6, y = 3}

Svar: Et eple koster 6 kr, en pære 3 kr.

Løs vs. NLøs:
  • Løs() gir eksakt svar (brøk, rot). Prøv dette først.
  • NLøs() gir numerisk svar (desimaltall). Bruk hvis Løs ikke klarer det.

Trigonometri og geometri

Arealformelen

A = ½ · a · b · sin(C)

Brukes når du kjenner to sider og vinkelen mellom dem (SVS). C er vinkelen mellom sidene a og b.

Cosinussetningen

c² = a² + b² − 2ab · cos(C)

Finner tredje side når du har to sider og vinkelen mellom (SVS). Kan også løses med hensyn på cos(C) for å finne en vinkel når du kjenner alle tre sidene.

Sinussetningen

a / sin(A) = b / sin(B) = c / sin(C)

Brukes når du har én side og to vinkler (eller to sider og en motstående vinkel). Her er a siden som står motsatt av vinkelen A osv.

Eksempel (sinussetningen)

Trekant ABC med A = 40°, B = 70° og a = 6. Finn b.

b / sin(70°) = 6 / sin(40°) → b = 6 · sin(70°) / sin(40°) ≈ 8.77

SSV-felle: Når du kjenner to sider og en ikke-mellomliggende vinkel, kan svaret være to ulike trekanter. Sjekk alltid at vinkelsummen blir 180° og at alle sider er positive.

Eksakte uttrykk

Til tentamen bør du beholde √-tegn og brøker når oppgaven sier «eksakt verdi». Ikke skriv 1.41 når du mener √2.

Eksempel (cosinussetningen, eksakt)

Trekant med a = 4, b = 5, C = 60°. Finn c.

c² = 16 + 25 − 2·4·5·cos(60°) = 41 − 40·½ = 21

c = √21 ≈ 4.58

Velg riktig løsning ut fra kontekst

Når du løser en andregradslikning får du ofte to svar. Spør deg selv:

  • Er størrelsen en lengde? → bare positive svar er gyldige.
  • Er det en tid? → bare t ≥ 0 er gyldig.
  • Er det en vinkel i trekant? → mellom 0° og 180°.
Skriv alltid begrunnelse: «Jeg velger x = 3 fordi lengder må være positive.» Dette gir tellende poeng.

Optimering – maksimum og minimum

Klassisk oppgave: du har en begrensning (f.eks. «50 m gjerde») og skal finne største mulige areal. Oppskrift:

Tegn en figur og sett navn på sidene (x, y, osv.).
Skriv begrensningen som likning. Eksempel: 2x + 2y = 50.
Skriv størrelsen som skal maksimeres (areal, volum) som funksjon.
Eliminer én variabel ved å bruke begrensningen: y = 25 − x.
Skriv alt som funksjon av én variabel: A(x) = x(25 − x).
Finn ekstremalpunkt:Ekstremalpunkt(A)
Bestem gyldighetsområde: for dette eksempelet må 0 < x < 25 (lengder positive).

Eksempel: rektangel med 20 m gjerde

Anne har 20 m gjerde og vil lage et rektangulært område med størst mulig areal.

Omkrets: 2x + 2y = 20 → y = 10 − x.

Areal: A(x) = x·(10 − x) = 10x − x².

A(x) := 10x - x^2 Ekstremalpunkt(A) → (5, 25)

Svar: Maksimalt areal er 25 m² når x = 5 m og y = 5 m (et kvadrat).

Gyldighetsområde: 0 < x < 10 (begge sider må være positive).

Overflate som funksjon av én variabel

Eksempel: en lukket boks (bunn og lokk) med kvadratisk bunn og volum 1000 cm³. Finn minimal overflate.

  1. Variabler: x = sidekant i bunn, h = høyde.
  2. Begrensning: x² · h = 1000 → h = 1000 / x².
  3. Overflate: O = 2x² + 4xh = 2x² + 4x · (1000/x²) = 2x² + 4000/x.
    Forklaring: 2x² er bunn + lokk, 4xh er de fire sideflatene.
  4. I GeoGebra: Ekstremalpunkt(O).
O(x) := 2x^2 + 4000/x Ekstremalpunkt(O) → (≈10, ≈600)

Tolkning: minste overflate ≈ 600 cm², ved sidekant x ≈ 10 cm.

Derivasjon og vekstfart

Tolkning av funksjonsverdier

I kontekst betyr f(x) alltid noe konkret. Tenk på hva variabelen står for.

Eksempel

V(t) = mengde vann (liter) i en tank etter t minutter.

  • V(0) = 0 betyr at tappingen starter fra tom tank ved t = 0.
  • V(5) = 40 betyr at det er 40 liter i tanken etter 5 minutter.
  • V'(5) = 8 betyr at vannet stiger med 8 liter per minutt i øyeblikket t = 5.

Gjennomsnittlig vs. momentan

Løs ulikheter med den deriverte

Spørsmål som «når vokser f raskere enn 10 enheter per tidsenhet?» blir til ulikheten f'(x) > 10.

f(x) := 2x^3 - 9x^2 + 12x f'(x) Løs(f'(x) > 0, x) → x < 1 eller x > 2

Hvis Løs ikke håndterer ulikheten, bruk NLøs() eller tegn f'(x) og se hvor grafen ligger over/under riktig verdi.

Sekantlinje i GeoGebra

For å vise gj.snittlig vekstfart visuelt, tegn linjen mellom to punkter:

A = (2, f(2)) B = (5, f(5)) sekant = Linje(A, B)

Tangent i et punkt

Tangenten er linjen som bare berører grafen i ett punkt – stigningstallet er f'(a).

f(x) := x^2 - 4x + 5 Tangent(3, f) → y = 2x − 4

Tolkning: stigningstallet 2 er det samme som f'(3) = 2·3 − 4 = 2. Tangenten i (3, 2) har likning y = 2x − 4.

Tips: Trenger du bare stigningstallet? Skriv f'(3). Trenger du hele likningen til tangenten? Bruk Tangent(3, f).

Vanlige CAS-feil

  • Glemt := ved definisjon. Riktig: f(x) := x^2. Feil: f(x) = x^2.
  • Glemt gangetegn: 2sin(30°) fungerer ikke. Skriv 2·sin(30°).
  • Glemt ° for vinkler: sin(60) gir radianer. Bruk sin(60°).
  • Glemt variabel i Løs: Løs(x+y=4, {x,y}) – variablene må stå til slutt.

Python-programmering

Variabler

En variabel er et navn som lagrer en verdi. Python gjetter selv om det er tall, tekst eller liste.

pris = 20 # heltall rente = 1.04 # desimaltall navn = "Per" # tekst (string) print(pris * 3) # → 60

For-løkke

En for-løkke gjentar koden et bestemt antall ganger. range(n) gir tallene 0, 1, 2, …, n−1.

# Summer tallene 1 til 10 sum = 0 for i in range(1, 11): sum = sum + i print(sum) # → 55
Hvordan lese en for-løkke: «For hver verdi i fra 1 til og med 10, gjør …». Indentering (innrykk) er viktig – Python forstår at koden hører til løkka fra innrykket.

While-løkke

En while-løkke kjører så lenge en betingelse er sann. Du vet ikke på forhånd hvor mange ganger den kjører.

# Hvor mange år før beløpet passerer 2000 kr? belop = 1000 ar = 0 while belop < 2000: belop = belop * 1.05 ar = ar + 1 print(ar) # → 15

Forskjell på for og while

ForWhile
Når du vet antall ganger på forhånd Når du vil kjøre til en betingelse er oppfylt
«Regn ut summen av 100 første tallene» «Regn til du passerer 1 million»
Ingen fare for evig løkke Kan gå evig hvis betingelsen aldri blir falsk

Geometriske rekker

En geometrisk rekke multipliserer med samme tall hvert ledd: a, a·k, a·k², a·k³, …

# Summen av første 10 ledd, a = 2, k = 3 a = 2 k = 3 s = 0 ledd = a for i in range(10): s = s + ledd ledd = ledd * k print(s) # → 59048

Figurmønster

Mønster: Figur 1 har 3 prikker, figur 2 har 5, figur 3 har 7, … Antall prikker i figur n?

# Skriv ut antall prikker i figur 1 til 10 for n in range(1, 11): prikker = 2*n + 1 print("Figur", n, ":", prikker, "prikker")

Tilpasse et program til et nytt problem

På tentamen får du ofte et ferdig program og skal endre det for et litt annet problem. Strategi:

  1. Les programmet linje for linje og skriv på norsk hva hver linje gjør.
  2. Finn hva som må endres: start-verdi, slutt-betingelse, regneregel?
  3. Bytt ut minst mulig – ikke skriv om hele programmet.
  4. Test med en liten verdi du kan regne i hodet for å sjekke at det stemmer.
Forklaring teller: Selv om koden er kort, skriv en setning om hva programmet gjør og hvorfor det løser oppgaven.
