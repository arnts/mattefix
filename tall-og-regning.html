1.1 Tall og tallmengder

Mengdeparenteser

Vi bruker krøllparenteser { } for å skrive mengder. Elementene i mengden listes opp med komma.

A = {1, 2, 3, 4, 5}

Vi skriver a ∈ A for «a er element i mengden A», og b ∉ A for «b er ikke element i A».

De viktigste tallmengdene

SymbolNavnBeskrivelseEksempler
N Naturlige tall Positive hele tall (telle-tall) 1, 2, 3, 4, 5, ...
Z Hele tall Naturlige tall + null + negative hele tall ..., -3, -2, -1, 0, 1, 2, 3, ...
Q Rasjonale tall Alle tall som kan skrives som en brøk a/b (der b ≠ 0) ½, -¾, 0.25, 3, -7
R Reelle tall Alle tall på tallinja (rasjonale + irrasjonale) π, √2, e, 0.333..., 7
Sammenheng: N ⊂ Z ⊂ Q ⊂ R. Det betyr at alle naturlige tall er hele tall, alle hele tall er rasjonale, osv. Irrasjonale tall (som π og √2) er reelle, men ikke rasjonale.

1.1 Eksponenter og potenser

En potens er en kompakt måte å skrive gjentatt multiplikasjon:

an = a · a · a · ... · a   (n faktorer)

Her er a grunntallet og n eksponenten.

Viktige potensregler

Multiplikasjon
am · an = am+n
23 · 24 = 27 = 128
Divisjon
am / an = am−n
56 / 52 = 54 = 625
Potens av potens
(am)n = am·n
(32)3 = 36 = 729
Potens av produkt
(a · b)n = an · bn
(2 · 3)3 = 23 · 33 = 216
Null-eksponent
a0 = 1   (a ≠ 0)
70 = 1,   x0 = 1
Negativ eksponent
a−n = 1 / an
2−3 = 1/8 = 0.125

1.1 Regneregler

Disse reglene gjelder for alle tall (og bokstaver som representerer tall):

Med tall

Kommutativ lov (addisjon)
a + b = b + a
3 + 5 = 5 + 3 = 8
Kommutativ lov (multiplikasjon)
a · b = b · a
4 · 7 = 7 · 4 = 28
Assosiativ lov
(a + b) + c = a + (b + c)
(2 + 3) + 4 = 2 + (3 + 4) = 9
Distributiv lov
a · (b + c) = a·b + a·c
3 · (4 + 5) = 12 + 15 = 27

Med bokstaver (algebra)

a(b + c) = ab + ac

Eksempel: 2x(3x + 4) = 6x2 + 8x

(a + b)2 = a2 + 2ab + b2

Eksempel: (x + 3)2 = x2 + 6x + 9

(a + b)(a − b) = a2 − b2

Eksempel: (x + 5)(x − 5) = x2 − 25

1.1 Regnerekkefølge

Når et uttrykk har flere operasjoner, regner vi i denne rekkefølgen:

Parenteser – Regn ut det som står inne i parenteser først. ( ... )
Potenser – Regn ut eksponenter. an
Multiplikasjon og divisjon – Fra venstre mot høyre. · og /
Addisjon og subtraksjon – Fra venstre mot høyre. + og −
Eksempel: 2 + 3 · 42 − (1 + 5)
= 2 + 3 · 16 − 6
= 2 + 48 − 6
= 44

1.2 Regning med brøk

Forkorting og utviding

Forkorte: Del teller og nevner med samme tall.

a · cb · c = ab

Eksempel: 1218 = 12 : 618 : 6 = 23

Utvide: Gang teller og nevner med samme tall.

Eksempel: 25 = 2 · 35 · 3 = 615

Addisjon og subtraksjon

Brøkene må ha felles nevner:

ac + bc = a + bc

Eksempel: 13 + 14 = 412 + 312 = 712

Multiplikasjon av brøk

ab · cd = a · cb · d

Eksempel: 23 · 57 = 1021

Divisjon av brøk

Vi snur den andre brøken og ganger:

ab : cd = ab · dc = a · db · c

Eksempel: 34 : 25 = 34 · 52 = 158

Bokstavregning med brøk

Samme regler gjelder med bokstaver:

ab + cd = ad + bcbd

Eksempel: 2x + 3x + 1 = 2(x + 1) + 3xx(x + 1) = 5x + 2x(x + 1)

1.3 Prosentregning

Andeler i brøk og prosent

Prosent betyr «per hundre». Vi regner om mellom brøk, desimaltall og prosent:

ab   →   a : b = desimaltall   →   desimaltall · 100 = prosent

Eksempel: 34 = 0,75 = 75 %

Sammenligning av to størrelser

Hvor mange prosent er a av b?

Prosentandel = ab · 100 %

Eksempel: 15 av 60 = 1560 · 100 % = 25 %

Prosentpoeng

Prosentpoeng er forskjellen mellom to prosenttall.

Eksempel: Renta gikk fra 3 % til 5 %. Det er en økning på 2 prosentpoeng.
Men selve renta økte med 23 · 100 % ≈ 66,7 % (prosentvis økning).

Prosentvis økning og reduksjon

Ny verdi = gammel verdi · (1 + p/100)   (økning)
Ny verdi = gammel verdi · (1 − p/100)   (reduksjon)

Eksempel: En vare koster 800 kr og øker med 15 %:
Ny pris = 800 · 1,15 = 920 kr

1.4 Vekstfaktor

Hva er vekstfaktor?

Vekstfaktoren er tallet vi ganger med for å gå fra gammel til ny verdi:

Ny verdi = gammel verdi · vekstfaktor

Økning og reduksjon

Økning med p %
Vekstfaktor = 1 + p100
Økning 20 % → vekstfaktor 1,20
Reduksjon med p %
Vekstfaktor = 1 − p100
Reduksjon 30 % → vekstfaktor 0,70

Regning med vekstfaktor

Vekstfaktorer kan ganges sammen for gjentatte endringer:

Samlet vekstfaktor = v1 · v2 · v3 · ...
Eksempel: En vare øker med 10 % og deretter reduseres med 20 %:
Samlet vekstfaktor = 1,10 · 0,80 = 0,88
Totalt er varen redusert med 12 % (ikke 10 %).

For gjentatt vekst med samme prosent over n perioder:

Ny verdi = startverdi · vekstfaktorn

Eksempel: Et beløp på 10 000 kr vokser med 5 % per år i 3 år:
10 000 · 1,053 = 10 000 · 1,157625 = 11 576,25 kr

Strekning, fart og tid

Sammenhengen mellom strekning (s), fart (v) og tid (t):

Strekning
s = v · t
Fart
v = st
Tid
t = sv

Enheter

m/s (meter per sekund)
SI-enheten for fart
Brukes i fysikk og naturfag
km/t (kilometer per time)
Vanlig i dagliglivet
Fartsgrenser, bilturer

Omregning mellom km/t og m/s

km/t → m/s:   del på 3,6
m/s → km/t:   gang med 3,6
Hvorfor 3,6? 1 km = 1000 m og 1 t = 3600 s. Så 1 km/t = 1000/3600 m/s = 1/3,6 m/s.

Eksempler

Strekning: En bil kjører i 80 km/t i 2,5 timer. Hvor langt? s = v · t = 80 · 2,5 = 200 km
Fart: En syklist sykler 45 km på 3 timer. Hva er farten? v = s / t = 45 / 3 = 15 km/t
Omregning: 90 km/t til m/s: 90 / 3,6 = 25 m/s
Omregning: 12 m/s til km/t: 12 · 3,6 = 43,2 km/t
Tid: En løper holder 8 m/s. Hvor lang tid bruker hun på 100 m? t = s / v = 100 / 8 = 12,5 s

1.5 Bevis og sammenhenger: Pytagoras' setning

I en rettvinklet trekant gjelder:

a2 + b2 = c2

Der c er hypotenusen (den lengste siden, mot den rette vinkelen) og a og b er de to katetene.

Bruk av Pytagoras

Finne hypotenusen: c = √(a2 + b2) a = 3, b = 4: c = √(9 + 16) = √25 = 5
Finne en katet: a = √(c2 − b2) c = 13, b = 5: a = √(169 − 25) = √144 = 12
Sjekke rettvinklet trekant: Stemmer a2 + b2 = c2? 3, 4, 5: 9 + 16 = 25 ✓ → Rettvinklet!

Bevisidé (geometrisk)

Tegn et stort kvadrat med side (a + b). Inne i dette kan vi lage fire kopier av trekanten og et lite kvadrat med side c. Da ser vi at:

(a + b)2 = 4 · ab2 + c2   ⇒   a2 + 2ab + b2 = 2ab + c2   ⇒   a2 + b2 = c2

1.6 Problemløsning

Pólyas fire steg for problemløsning:

1

Forstå problemet

Hva er gitt? Hva skal vi finne? Tegn en figur eller lag en liste over det du vet.

2

Lage en plan

Hvilken metode passer? Har du sett et lignende problem? Kan du dele det opp?

3

Gjennomføre planen

Regn steg for steg. Sjekk hvert steg underveis. Vær nøyaktig.

4

Se tilbake

Er svaret rimelig? Kan du kontrollere det? Kan metoden brukes på andre problemer?

Eksempel: Problemløsning

Oppgave: En rektangulær hage er 12 m lang og 5 m bred. Hvor lang er diagonalen?
Forstå: Rektangel med sider 12 m og 5 m. Diagonalen deler rektangelet i to rettvinklede trekanter.
Plan: Bruk Pytagoras' setning der a = 12 og b = 5.
Gjennomfør: d = √(122 + 52) = √(144 + 25) = √169 = 13 m
Se tilbake: 132 = 169 = 144 + 25 ✓ Svaret er rimelig (diagonalen er lenger enn begge sidene).
